Utilizando PHP e MySQL, crie uma página web onde você consiga:
1. CADASTRAR, ALTERAR e EXCLUIR um médico
2. LISTAR todos os médicos

Informações necessárias:
- Nome do médico
- CRM
- Telefone
- Especialidade: (Ex.: ANESTESIOLOGIA, FONOAUDIOLOGIA, ENDOCRINOLOGIA, GERIATRIA, HEMATOLOGIA, CIRURGIA GERAL)

-Diferenciais (Opcionais):
- utilizar POO (Programação Orientada a Objetos)
- Enviar os comandos SQL de criação de tabelas, com toda a estruturas das tabelas utilizadas.
- Utilizar JavaScript

Não é necessário o sistema estar on-line. Envie apenas os arquivos com o código. Pode ser uma pasta zipada ou um link do github.

Seja criativo!
