Utilizando PHP e MySQL, crie uma página web onde você consiga:
1. CADASTRAR, ALTERAR e EXCLUIR um médico
2. LISTAR todos os médicos

Informações necessárias:
- Nome do médico
- CRM
- Telefone
- Especialidade: (Ex.: ANESTESIOLOGIA, FONOAUDIOLOGIA, ENDOCRINOLOGIA, GERIATRIA, HEMATOLOGIA, CIRURGIA GERAL)

Utilizado:
PHP
Mysql
Bootstrap
JavaScript
Css
